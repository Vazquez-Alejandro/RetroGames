# RetroGames 🕹️

Pre-entrega de ecommerce - Talento Tech React JS, 1° cuatrimestre 2026.

**RetroGames** es una tienda online de juegos, consolas y accesorios retro, hecha con React.

## Para correrlo

```bash
npm install
npm run dev
```

Después abrí el http://localhost que te aparezca.
